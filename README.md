# convert_gas_flow_from_std

## Описание 
Программа позволяет быстро и точно пересчитать расход газа из стандартных условий в рабочие (без учета коэффициента сжимаемости) на основе входных параметров (расход в стандартных условиях, давление и температура в рабочих условиях). Предназначена для инженеров‑проектировщиков, технологов и метрологов.
